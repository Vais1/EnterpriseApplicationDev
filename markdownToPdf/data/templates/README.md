# Templates Directory

This directory contains saved Markdown templates.

## Usage

Templates are automatically saved to browser localStorage when you use:
- Keyboard shortcut: `Ctrl+Shift+S` (or `Cmd+Shift+S` on Mac)
- Enter a name for your template when prompted

## Storage

Templates are stored in the browser's localStorage under the key `markdown_templates`.

Each template includes:
- Template name
- Markdown content
- Timestamp of creation

## Exporting Templates

To export templates for backup or sharing:
1. Open browser DevTools (F12)
2. Go to Application/Storage tab
3. Find localStorage
4. Export the `markdown_templates` key

You can also manually save your markdown files using the "Load .md File" feature.
